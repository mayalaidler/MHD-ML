**Machine Learning Acceleration of MHD Simulations**

This repository contains the code and analysis for my senior thesis, which investigates how machine learning can approximate outcomes of large-scale magnetohydrodynamic (MHD) simulations to reduce computational cost.

**Overview**

Magnetohydrodynamic simulations model the evolution of physical quantities such as gas density, velocity fields, magnetic field strength, and turbulent energy across three-dimensional grids. These simulations are computationally expensive and produce terabyte-scale outputs when run on high-performance computing (HPC) systems.

The goal of this project is to train machine learning models to predict key simulation features from initial conditions, reducing the need to rerun expensive simulations when exploring parameter space.
